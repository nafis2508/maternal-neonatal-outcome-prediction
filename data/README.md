# Dataset Information

## Source

The dataset contains anonymized healthcare facility delivery records collected from Kenya and Uganda.

The data was originally published for research purposes and has been used in this project to analyze and predict maternal and neonatal birth outcomes using machine learning techniques.

## Dataset Contents

The dataset includes variables related to:

- Birth outcomes
- Birth weight
- Gestational age
- APGAR scores
- Maternal characteristics
- Delivery conditions
- Healthcare facility information
