# Maternal & Neonatal Outcome Prediction using Machine Learning

## Project Overview

This project applies machine learning techniques to predict maternal and neonatal birth outcomes using healthcare facility records collected from Kenya and Uganda.

The analysis explores how maternal characteristics, delivery conditions, gestational age, birth weight, APGAR scores, and clinical indicators influence pregnancy and birth outcomes.

The project evaluates multiple machine learning algorithms and compares their predictive performance for healthcare outcome classification.

---

## Business Problem

Healthcare providers require reliable methods to identify factors associated with adverse pregnancy and birth outcomes.

This project demonstrates how machine learning can support data-driven decision-making in maternal and neonatal healthcare.

---

## Dataset Information

Dataset Size: 61,018 records

Source: Kenya and Uganda healthcare facility delivery records

Key Variables:

- APGAR Scores
- Birth Weight
- Gestational Age
- Mode of Delivery
- Maternal Status
- Baby Status
- Birth Outcome
- Delivery Facility Information

## Key Findings

- Random Forest achieved the highest predictive performance among the evaluated models.
- Birth outcomes were strongly influenced by gestational age, birth weight, and APGAR scores.
- Feature engineering and preprocessing significantly improved model performance.
- Machine learning demonstrated potential for supporting healthcare decision-making in resource-constrained environments.
